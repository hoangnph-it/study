WHAT IS A STYLE REFERENCE?

A style reference is a source for visual aesthetics. It tells MidJourney to extract elements like color palette, medium, texture, and overall style — then apply them to your results.

🖼️
Images
Upload or drag an image into the style reference box
🔢
SREF Codes
Numbers linked to styles in MidJourney's Style Space (4B+ codes)

USING IMAGES AS STYLE REFERENCES

1
Drag & Drop
Drag an image to the prompt bar → drop in the style reference box → type prompt → submit
2
Image URL
Type prompt, add --sref, paste the image URL
3
Uploaded Images
Access your uploaded images → select "style reference" → click the image

BLENDING MULTIPLE STYLES

Multiple Images
Add more images to the style reference box. MidJourney blends their aesthetics into one cohesive style.
Multiple SREF Codes
Separate codes with a space after --sref. Styles are blended into a new combined style.
WEIGHTING INDIVIDUAL REFERENCES
--sref 123456::2 789012::1
Use ::number after each code to set relative weight. Image weights only work on Discord.

⚠️ STYLE VERSION --sv — CRITICAL TO KNOW

The same SREF code can produce completely different styles depending on which model you use.

SV4
Default for V6 models
Also used by Niji 6
SV6
Default for V7 models
Used by Style Explorer

🔑 To use old V6 SREF codes in V7 prompts, add --sv 4 to your prompt.

ADVANCED TECHNIQUES

1
Push Beyond Base Style
Be more direct and specific in your prompt text. Instead of "a man", try "photo of a man" to override an illustration-style SREF.
2
Use --raw for Prompt Adherence
Improves how closely results match your prompt text. Reduces MidJourney's default aesthetic styling.
3
Layer --exp Parameter
Introduces more detail and shifts colors. Can create muted, mustardy tones and desaturated palettes.
4
Add Personalization
Your personalization profile + SREF creates another customization layer. Lower style weight gives personalization more room.
5
Mood Boards for Diversity
Unlike SREFs that blend into one style, mood boards preserve variety across images while maintaining a cohesive look.

STYLE REFERENCES IN NIJI 6

ANIME MODEL Add --niji to your prompt
V6 SREF codes work with Niji 6 (both use SV4 system)
Newer V7/Style Explorer codes won't match on Niji 6
Workaround: Create an image with the V7 SREF, then use that image as a style reference in Niji

WORKFLOW & ORGANIZATION TIPS

🏷️ Custom Aliases
Create shortcuts on Discord with /prefer option set. Use --alias-name in prompts. Works on website too.
🔍 Search by SREF
On the Organize page, type --sref in search to filter images by style reference. Hover to preview.
❤️ Style Explorer Favorites
Heart styles in the explorer to save them. View all favorites by clicking "Likes". Search by keyword.
📋 List All Shortcuts
Type /prefer option list on Discord to see all your saved aliases and shortcuts.

WHEN RESULTS DON'T MATCH YOUR VISION

Follow this order to improve prompt adherence:

① Lower style weight — start here first
② Be more specific in your prompt text
③ Add --raw for better prompt adherence
④ Lower stylize value to reduce default aesthetics
⑤ Adjust other parameters or modify prompt text

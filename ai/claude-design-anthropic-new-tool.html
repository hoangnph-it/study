THE 3-STEP SETUP FLOW

1
BUILD YOUR DESIGN SYSTEM
Provide a company name, blurb, GitHub repo, brand guidelines, and logo. Claude ingests everything and drafts colors, gradients, neutrals, type, buttons, badges, and cards. Review and approve each piece.
2
CREATE A PROJECT WITH CONTEXT
Pick prototype, slides, or one-pager. Attach the design system, screenshots, or a codebase. For slides, drop a PDF — Claude invokes a read-PDF skill, plans the deck (e.g. 19 slides), then generates.
3
ITERATE, EXPORT, OR HAND OFF
Tweak panel for quick edits, comment on specific elements, draw on the canvas, or edit directly. Present fullscreen in-tab, or export anywhere when ready.

CONTEXT IS THE PRODUCT
Anthropic isn't copying Canva or Gamma — it's absorbing them.
When design, slides, and code all live in the same ecosystem, you stop ferrying context between tools. A "planning phase up front" costs more tokens once, but saves far more by not chasing the wrong path. Claude Design is Co-work, iterated for visuals.
